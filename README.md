# 💰 Finper - Gestão de Finanças Pessoais

Aplicativo mobile desenvolvido em **Flutter** para auxiliar usuários no controle de suas finanças pessoais, permitindo o gerenciamento de receitas, despesas, metas financeiras e acompanhamento da saúde financeira de forma simples e intuitiva.

## 📱 Sobre o Projeto

O **FinanTrack** foi criado com o objetivo de ajudar usuários a organizarem suas finanças, registrando movimentações financeiras e visualizando relatórios que auxiliam na tomada de decisões.

Com uma interface moderna e responsiva, o aplicativo busca proporcionar uma experiência prática para o controle financeiro diário.

---

## ✨ Funcionalidades

- ✅ Registro de receitas e despesas
- ✅ Categorias personalizadas para transações
- ✅ Visualização do saldo atual
- ✅ Histórico de movimentações
- ✅ Relatórios e gráficos financeiros
- ✅ Filtros por período
- ✅ Armazenamento seguro dos dados
- ✅ Interface responsiva para Android e iOS

## 🚀 Como Executar o Projeto

### Pré-requisitos

- Flutter SDK instalado
- Android Studio ou VS Code
- Emulador Android/iOS ou dispositivo físico
- Conta Firebase configurada (opcional)

### Clone o repositório

```bash
git clone https://github.com/seu-usuario/finantrack.git
```

### Acesse a pasta do projeto

```bash
cd finper
```

### Instale as dependências

```bash
flutter pub get
```

### Execute o aplicativo

```bash
flutter run
```
